Excel-01-Advanced-Lookups
Overview

This repository documents my 14-day learning journey through Advanced Lookup Functions in Microsoft Excel.

The goal of this bootcamp is to move beyond basic spreadsheet usage and develop practical lookup skills used in real-world business reporting, data analysis, operations management, finance, HR, and dashboard development.

Throughout this bootcamp, I explore traditional lookup methods such as VLOOKUP, modern alternatives like XLOOKUP, and advanced techniques using INDEX and MATCH.

Learning Objectives

By completing this bootcamp, I aim to:

Understand the fundamentals of lookup logic
Master VLOOKUP and its limitations
Learn XLOOKUP for modern Excel workflows
Handle missing values and lookup errors
Perform reverse lookups
Use INDEX and MATCH effectively
Build two-way lookup systems
Create multiple-criteria lookups
Develop dynamic lookup dashboards
Apply lookup techniques in practical business scenarios
