Excel Advanced Lookups Bootcamp

A beginner-friendly Excel practice repository designed to help learners master lookup functions through hands-on exercises and real-world business scenarios.

📚 Topics Covered

Week 1

* Understanding Lookup Logic
* VLOOKUP Basics
* VLOOKUP Limitations
* XLOOKUP
* Error Handling
* Reverse Lookup
* Mini Project

Week 2

* INDEX
* MATCH
* INDEX + MATCH
* Two-Way Lookup
* Multiple Criteria Lookup
* Dynamic Lookup Dashboard
* Final Project

🛠 Prerequisites

* Microsoft Excel 365 (Recommended)
* Microsoft Excel 2021 or later
* Basic understanding of Excel worksheets and tables

🚀 Getting Started

Step 1: Open the Day 1 Workbook

Start with:

```text
Day01_Concepts.xlsx
```

Learn the fundamentals of lookup logic and how data retrieval works in Excel.

Step 2: Complete Daily Exercises

Work through each workbook in sequence:

```text
Day02_VLOOKUP.xlsx
Day03_VLOOKUP_Limitations.xlsx
Day04_XLOOKUP.xlsx
...
```

Each file contains practical examples and exercises focused on a specific lookup technique.

Step 3: Build the Projects

Complete:

```text
Day07_Mini_Project.xlsx
Day14_Final_Project.xlsx
```

These projects combine multiple lookup functions into real-world business solutions.

📊 Dataset

The same employee and sales datasets are reused throughout the bootcamp to simulate real-world reporting scenarios.

Example fields include:

* Employee ID
* Employee Name
* Department
* Salary
* Product Name
* Monthly Sales
* Region

Using a consistent dataset allows learners to focus on mastering lookup techniques rather than understanding new data structures each day.

🎯 Learning Goals

By completing this bootcamp, learners will be able to:

* Use VLOOKUP confidently
* Understand the limitations of VLOOKUP
* Master XLOOKUP
* Perform reverse lookups
* Use INDEX and MATCH together
* Build two-way lookups
* Create multiple-criteria lookups
* Handle lookup errors effectively
* Build dynamic lookup dashboards
* Apply lookup techniques to real business problems
