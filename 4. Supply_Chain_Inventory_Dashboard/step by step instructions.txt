STEP 7 — CREATE CALCULATED COLUMNS
A. INVENTORY VALUE

In Inventory Data

Create column:

Inventory Value

Formula:

=[@[Current Stock]]*[@[Unit Cost]]
B. STOCK STATUS

Create column:

Stock Status

Formula:

=IF([@[Current Stock]]=0,"Stock-Out",
IF([@[Current Stock]]<[@[Reorder Level]],"Low Stock",
IF([@[Current Stock]]>[@[Reorder Level]]*3,"Overstock","Normal")))
C. AVERAGE DAILY SALES

Create column:

Average Daily Sales

Enter manually:

Product ID	Average Daily Sales
P001	8
P002	5
P003	2
P004	3
P005	20
P006	1
P007	25
P008	2
P009	1
P010	6
D. LEAD TIME

Create column:

Lead Time

Formula:

=XLOOKUP([@Supplier],tblSuppliers[Supplier],tblSuppliers[Lead Time Days])

Older Excel:

=VLOOKUP([@Supplier],tblSuppliers,4,FALSE)
E. RECOMMENDED REORDER POINT

Create column:

Recommended Reorder Point

Formula:

=[@[Average Daily Sales]]*[@[Lead Time]]
STEP 8 — ORDERS DATA CALCULATIONS
A. DELAY DAYS

Create column:

Delay Days

Formula:

=[@[Actual Delivery]]-[@[Expected Delivery]]

IMPORTANT:

After formula:

Select Delay Days column
Change format from Date to:
General

Otherwise Excel shows:

01-Jan-00
#######
00-Jan-00
B. DELIVERY STATUS

Create column:

Delivery Status

Formula:

=IF([@[Delay Days]]>0,"Delayed","On Time")
C. SUPPLIER COLUMN

Orders Data does not contain supplier names.

Create new column:

Supplier

Formula:

=XLOOKUP([@[Product ID]],tblInventory[Product ID],tblInventory[Supplier])

Older Excel:

=VLOOKUP([@[Product ID]],tblInventory,4,FALSE)

Yellow

STEP 12 — KPI CARDS

Create these in Dashboard sheet.

TOTAL INVENTORY VALUE
=SUM(tblInventory[Inventory Value])
TOTAL PRODUCTS
=COUNTA(tblInventory[Product ID])
LOW STOCK ITEMS
=COUNTIF(tblInventory[Stock Status],"Low Stock")
OVERSTOCK ITEMS
=COUNTIF(tblInventory[Stock Status],"Overstock")
DELAYED ORDERS
=COUNTIF(tblOrders[Delivery Status],"Delayed")
